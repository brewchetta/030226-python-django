# Mini Project

## Deliverables

- You are encouraged to build user stories before you start

- Create a new stand-alone project in Django

- You should have at least 2 views/pages

- You should use context at least once

- You should use a `base.html`

## Example User Stories

- DONE - user will be able to see featured videos on the homepage

- user will be able to navigate to other pages via the navbar

- DONE - user will be able to read our mission statement on the about page

- user will be able to see an index for all the videos and click a video to go to that video's detail page

## Pages to Build

- homepage
- about page
- detail page
- videos index page
