# Day Five

## Deliverables

- Generate user stories
- Generate a new project and app
- Add a homepage and `base.html`
- Create at least one model
- Create full CRUD on at least one model
- BONUS: Create a second model with an association to the first one

## Example Project:

This is a project you can build if you're not sure what you want to do...

Build a superhero registration database. You will be able to fill out a form to register a new
superhero, see all superheros, update a superhero, and delete one from the database.

As a bonus create an additional Superpower model which allows you to track which superheroes
have which superpowers (the exact implementation is up to you...)
